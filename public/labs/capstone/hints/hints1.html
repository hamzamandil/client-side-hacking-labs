ShopEZ - Hints

Only reveal what you need. Try to solve it yourself first.

Focus on the search functionality. What happens when you search for something? Look at the URL and how the results are displayed on the page.
The search query comes from the ?q= URL parameter. Try searching for something and watch the URL change. The query is sent to an API and the response is rendered on the page.
The API returns a field called resultHtml that contains your search query embedded in HTML. This field is passed through a "sanitizer" function, then rendered via innerHTML. Open DevTools and read the JavaScript to find the sanitizer.
The sanitizer removes <script> tags, <iframe, and javascript: — but it doesn't remove event handler attributes like onerror, onfocus, or onmouseover. The API reflects your query directly into the HTML without server-side escaping.
Search for an HTML tag with an event handler that the sanitizer doesn't block. The payload goes in the ?q= parameter, flows through the API's resultHtml response field, passes the client-side sanitizer, and gets rendered via innerHTML.
